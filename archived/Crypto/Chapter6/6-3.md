需要的东西齐了，编写脚本读取每行解密，记得删除 data.txt 的前两行

```python
import gmpy2


n = 920139713

e = 19

# p、q 在线网站分解得到
p = 18443

q = 49891

phi = (p - 1) * (q - 1)

d = gmpy2.invert(e, phi)

result = []

with open("../02c01a13-3a86-47de-8648-f03328a5e5d8/RsaRoll/data.txt", "r") as f:
	for line in f.readlines():
	line = line.strip('\n')

	result.append(chr(pow(int(line), d, n)))

for i in result:
	print(i, end='')
```
