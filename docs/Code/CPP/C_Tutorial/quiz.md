# C 测验

## 选择题

### 1. C 程序入口函数通常是什么？

A. `start`  
B. `main`  
C. `run`  
D. `init`  

答案：B

### 2. 哪个函数用于动态分配内存？

A. `malloc`  
B. `printf`  
C. `strlen`  
D. `fclose`  

答案：A

### 3. C 字符串以什么结尾？

A. `\n`  
B. `\0`  
C. `EOF`  
D. `NULL` 指针  

答案：B

### 4. 数组越界访问属于什么？

A. 良定义行为  
B. 实现定义行为  
C. 未定义行为  
D. 预处理行为  

答案：C

4. 使用结构体保存学生姓名和成绩。
5. 使用 `malloc` 创建动态数组并正确释放。
