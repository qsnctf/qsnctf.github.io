# C++ 教程

C++ 是一门支持过程式、面向对象、泛型和系统级编程的语言。它既能直接控制内存和对象生命周期，也提供了标准模板库（STL）、智能指针、线程库等现代抽象能力。

本教程面向初学者和 CTF/工程实践学习者，建议使用 C++17 或更新标准进行练习。

## 编译示例

```bash
c++ -std=c++17 -Wall -Wextra -Wpedantic main.cpp -o main
./main
```

Windows 下使用 MSVC：

```powershell
cl /std:c++17 /EHsc main.cpp
main.exe
```
