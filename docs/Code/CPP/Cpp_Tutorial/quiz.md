# C++ 测验

## 选择题

### 1. C++ 标准输出通常使用哪个对象？

A. `std::cin`  
B. `std::cout`  
C. `std::cerr`  
D. `std::ifstream`  

答案：B

### 2. `std::vector` 的特点是什么？

A. 固定长度数组  
B. 动态数组  
C. 有序映射  
D. 线程对象  

答案：B

### 3. 哪个关键字常用于定义不可修改对象？

A. `const`  
B. `goto`  
C. `delete`  
D. `throw`  

答案：A

### 4. 抽象类通常包含什么？

A. 纯虚函数  
B. 宏定义  
C. 全局变量  
D. 文件流  

答案：A

## 实操题

1. 编写程序读取一组整数并输出最大值。
2. 使用 `std::map` 统计字符串出现次数。
3. 编写一个 `Point` 类，包含 `x`、`y`，并重载 `operator+`。
4. 使用 `std::ifstream` 逐行读取文件。
