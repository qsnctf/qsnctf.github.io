# HTML 学习目录

HTML 是描述网页结构与语义的标记语言。这个目录按教程、HTML5、媒体和参考手册拆分，适合从零学习，也适合作为 Web 与 CTF 基础速查。

## 学习顺序

1. [HTML 教程](tutorial/index.md)：先掌握元素、属性、文本、链接、表格、表单、颜色、脚本和 URL。
2. [HTML5](html5/index.md)：再学习现代 HTML 平台能力，包括语义元素、Canvas、SVG、表单增强、存储、Worker 和实时通信。
3. [HTML 媒体](media/index.md)：集中理解音频、视频、插件历史和媒体实例。
4. [HTML 参考手册](reference/index.md)：最后用标签、属性、事件、实体、字符集、HTTP 等表格做检索。

## 现代边界

- 现代网页默认使用 `<!doctype html>` 与 UTF-8。
- “HTML5”在课程中作为学习分组，现代规范实际持续演进。
- Web SQL、Application Cache、Flash、Java Applet、`frameset` 等内容只作历史和兼容说明，不应用于新项目。
- 表单客户端验证、隐藏字段和前端脚本不能替代服务端验证与授权。
- 学习 XSS、URL 编码、实体和 HTTP 时，只在本地、靶场或明确授权环境中实验。
