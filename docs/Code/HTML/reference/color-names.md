# CSS 颜色名参考

命名颜色是 CSS `<named-color>` 关键字，可用于 HTML 元素的 CSS 样式。
名称不区分 ASCII 大小写；本页使用小写并按色系紧凑列出标准常见集合。

## 基本用法

```css
.notice {
  color: navy;
  background-color: aliceblue;
  border: 2px solid steelblue;
}
```

## 特殊关键字与限制

- `transparent` 表示完全透明的黑色，不是普通固定不透明颜色。
- `currentColor` 取元素计算后的 `color` 值，属于颜色关键字但不是命名色。
- `canvas`、`canvastext` 等系统颜色会跟随用户代理和系统主题，不在上表固定色值中。
- 同义名包括 `aqua`/`cyan`、`fuchsia`/`magenta` 以及多组 `gray`/`grey`。
- 名称不能说明可访问性；必须测量实际字号、字重、前景、背景和交互状态的对比度。
- 不只用颜色传达错误、成功或选中状态，应同时提供文字、图标或形状。

需要用户选择颜色时参见[拾色器](color-picker.md)。精确品牌色通常用设计令牌和十六进制、RGB、HSL 或 OKLCH。
