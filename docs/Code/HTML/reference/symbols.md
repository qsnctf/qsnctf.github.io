# HTML 常用符号参考

现代 HTML 以 Unicode 和 UTF-8 为默认，常用符号通常可以直接写入源码。
字符引用适合有 HTML 语法含义、难输入或需要明确源码形式的字符。

## 排版与法律符号

| 字符 | Unicode | 推荐引用 | 含义 |
| :---: | --- | --- | --- |
| © | `U+00A9` | `&copy;` | 版权 |
| ® | `U+00AE` | `&reg;` | 注册商标 |
| ™ | `U+2122` | `&trade;` | 商标 |
| § | `U+00A7` | `&sect;` | 章节 |
| ¶ | `U+00B6` | `&para;` | 段落标记 |
| … | `U+2026` | `&hellip;` | 省略号 |
| – | `U+2013` | `&ndash;` | en dash，范围或连接 |
| — | `U+2014` | `&mdash;` | em dash，插入语 |
| “ ” | `U+201C/201D` | `&ldquo;` / `&rdquo;` | 英文双弯引号 |
| ‘ ’ | `U+2018/2019` | `&lsquo;` / `&rsquo;` | 英文单弯引号 |
| NBSP | `U+00A0` | `&nbsp;` | 不换行空格 |

排版符号应符合内容语言。不要用多个 `&nbsp;`、破折号或点号模拟布局。

## 货币符号

| 字符 | Unicode | 引用 | 货币或用途 |
| :---: | --- | --- | --- |
| ¥ | `U+00A5` | `&yen;` | 日元或人民币，需结合货币代码 |
| $ | `U+0024` | `&#36;` | 美元等多种 dollar 货币 |
| € | `U+20AC` | `&euro;` | 欧元 |
| £ | `U+00A3` | `&pound;` | 英镑 |
| ¢ | `U+00A2` | `&cent;` | 分币 |
| ₹ | `U+20B9` | `&#x20B9;` | 印度卢比 |
| ₩ | `U+20A9` | `&#x20A9;` | 韩元 |
| ₽ | `U+20BD` | `&#x20BD;` | 俄罗斯卢布 |

金额数据应同时保存明确的 ISO 4217 货币代码，例如 `CNY 100.00`，不能只看符号。

## 数学与逻辑符号

| 字符 | Unicode | 引用 | 含义 |
| :---: | --- | --- | --- |
| ± | `U+00B1` | `&plusmn;` | 正负 |
| × | `U+00D7` | `&times;` | 乘 |
| ÷ | `U+00F7` | `&divide;` | 除 |
| − | `U+2212` | `&minus;` | 数学负号，不同于连字符 `-` |
| ≠ | `U+2260` | `&ne;` | 不等于 |
| ≤ | `U+2264` | `&le;` | 小于等于 |
| ≥ | `U+2265` | `&ge;` | 大于等于 |
| ≈ | `U+2248` | `&asymp;` | 约等于 |
| ∞ | `U+221E` | `&infin;` | 无穷 |
| √ | `U+221A` | `&radic;` | 平方根 |
| ∑ | `U+2211` | `&sum;` | 求和 |
| ∏ | `U+220F` | `&prod;` | 连乘 |
| ∈ | `U+2208` | `&isin;` | 属于 |
| ∉ | `U+2209` | `&notin;` | 不属于 |
| ∧ | `U+2227` | `&and;` | 逻辑与 |
| ∨ | `U+2228` | `&or;` | 逻辑或 |

复杂公式宜使用 MathML 或项目采用的公式渲染方案，并提供可访问文本。

## 箭头与界面符号

| 字符 | Unicode | 引用 | 常见用途 |
| :---: | --- | --- | --- |
| ← | `U+2190` | `&larr;` | 向左 |
| ↑ | `U+2191` | `&uarr;` | 向上 |
| → | `U+2192` | `&rarr;` | 向右 |
| ↓ | `U+2193` | `&darr;` | 向下 |
| ↔ | `U+2194` | `&harr;` | 双向关系 |
| ✓ | `U+2713` | `&#x2713;` | 通过或完成 |
| ✗ | `U+2717` | `&#x2717;` | 失败或取消 |
| ★ | `U+2605` | `&#x2605;` | 星标 |

界面图标不能只依赖视觉形状传达含义；按钮仍需可访问名称，状态仍需文字说明。

## 使用检查

- 文件和 HTTP 响应统一为 UTF-8，避免依赖旧编码覆盖范围。
- `<`、`&` 等 HTML 语法字符按上下文转义，参见[字符实体](entities.md)。
- 区分连字符 `-`、负号 `−`、en dash `–` 和 em dash `—`。
- 区分字母 `x`、乘号 `×`，以及普通空格和不换行空格。
- 搜索、用户名和安全比较前考虑 Unicode 规范化与同形字符风险。
