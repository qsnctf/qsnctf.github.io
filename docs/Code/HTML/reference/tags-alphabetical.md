# HTML 标签：字母排序
本页列出工程中常见的现代 HTML 元素，便于按名称检索。
它不是 HTML Living Standard 的穷尽清单；较少使用或实验性元素应另查规范与兼容性数据。

## 现代常用元素
| 元素（字母顺序） | 用途 | 状态或注意 |
| --- | --- | --- |
| `a` | 超链接或同页片段链接 | 现代；有 `href` 时才具有链接语义 |
| `abbr` | 缩写或简称 | 现代；正文仍应能理解其含义 |
| `address` | 最近 `article` 或页面的联系信息 | 现代；不是任意邮政地址容器 |
| `area` | 图像映射中的可点击区域 | 现代；可点击区域需要替代文本 |
| `article` | 可独立分发或复用的内容 | 现代 |
| `aside` | 与主内容间接相关的补充内容 | 现代 |
| `audio` | 音频播放器 | 现代；提供控件和文字稿 |
| `b` | 引起注意但不增加重要性的文本 | 现代；不要仅为加粗而使用 |
| `base` | 设置文档相对 URL 的基准 | 现代；影响全页 URL，谨慎使用 |
| `bdi` | 隔离未知方向的双向文本 | 现代 |
| `bdo` | 强制覆盖文本方向 | 现代；仅在明确需要时使用 |
| `blockquote` | 块级引用 | 现代；`cite` 属性不会自动显示来源 |
| `body` | 文档内容主体 | 现代；每个文档一个 |
| `br` | 语义上的换行 | 现代；不用来制造段落间距 |
| `button` | 可操作按钮 | 现代；表单内显式写 `type` |
| `canvas` | 脚本驱动的位图画布 | 现代；必须提供等价替代 |
| `caption` | 表格标题 | 现代；作为 `table` 的首个子元素 |
| `cite` | 作品标题 | 现代；不是通用的人名标签 |
| `code` | 计算机代码片段 | 现代 |
| `col` / `colgroup` | 表格列及列分组 | 现代；`col` 位于 `colgroup` 中 |
| `data` | 内容与机器可读值关联 | 现代；时间值优先 `time` |
| `datalist` | 为输入框提供建议选项 | 现代；界面和辅助技术支持有差异 |
| `dd` / `dt` | 描述列表中的描述与名称 | 现代 |
| `del` | 已删除内容 | 现代；可带 `datetime` |
| `details` | 可展开的披露区域 | 现代；标题使用 `summary` |
| `dfn` | 术语的定义实例 | 现代 |
| `dialog` | 对话框或模态窗口 | 现代；模态使用 `showModal()` |
| `div` | 无额外语义的通用容器 | 现代；有合适语义元素时不要替代 |
| `dl` | 名称与值组成的描述列表 | 现代 |
| `em` | 语气强调 | 现代；不同于视觉斜体 |
| `embed` | 嵌入外部内容 | 现代；插件式内容兼容性有限 |
| `fieldset` | 表单控件分组 | 现代；配合 `legend` |
| `figure` / `figcaption` | 可独立引用的内容及说明 | 现代 |
| `footer` | 页面或章节的页脚 | 现代 |
| `form` | 提交控件集合 | 现代；服务端必须验证 |
| `h1`-`h6` | 六级章节标题 | 现代；按内容层级选择 |
| `head` | 文档元数据容器 | 现代；每个文档一个 |
| `header` | 页面或章节的引导内容 | 现代 |
| `hgroup` | 标题与副标题分组 | 现代；使用场景较少 |
| `hr` | 段落级主题转换 | 现代；不是纯装饰线 |
| `html` | 文档根元素 | 现代；设置 `lang` |
| `i` | 不同语态、术语或分类文本 | 现代；不要仅为斜体而使用 |
| `iframe` | 嵌入另一个浏览上下文 | 现代；设置 `title` 和最小权限 |
| `img` | 嵌入图片 | 现代；需要合适 `alt` 与尺寸 |
| `input` | 多类型表单输入控件 | 现代；需要标签和 `name` |
| `ins` | 已插入内容 | 现代；可带 `datetime` |
| `kbd` | 用户输入内容 | 现代 |
| `label` | 表单控件标签 | 现代；用 `for` 关联控件 `id` |
| `legend` | `fieldset` 的标题 | 现代 |
| `li` | 列表项 | 现代；用于 `ol`、`ul` 或 `menu` |
| `link` | 关联样式表、图标等外部资源 | 现代；通常位于 `head` |
| `main` | 文档独有的主要内容 | 现代；通常只有一个可见主地标 |
| `map` | 定义图像映射 | 现代 |
| `mark` | 因当前上下文而高亮的文本 | 现代；保证对比度 |
| `menu` | 命令或项目列表 | 现代；通常表现类似 `ul` |
| `meta` | 字符集、视口等元数据 | 现代；通常位于 `head` |
| `meter` | 已知范围内的标量测量 | 现代；不要表示普通进度 |
| `nav` | 主要导航链接组 | 现代；不必包围每组链接 |
| `noscript` | 脚本不可用时的替代内容 | 现代；内容模型依上下文变化 |
| `object` | 嵌入外部资源 | 现代；复杂内容优先更专用元素 |
| `ol` | 有序列表 | 现代 |
| `optgroup` | `select` 选项分组 | 现代；使用 `label` |
| `option` | 下拉或建议列表选项 | 现代 |
| `output` | 计算或操作结果 | 现代；可用 `for` 关联输入 |
| `p` | 段落 | 现代；不能包含块级流内容 |
| `picture` | 响应式图片源选择 | 现代；最终包含 `img` |
| `pre` | 保留空白的预格式文本 | 现代；长内容注意横向滚动 |
| `progress` | 任务完成进度 | 现代；不确定进度时省略 `value` |
| `q` | 行内引用 | 现代；浏览器通常自动加引号 |
| `ruby` / `rt` / `rp` | 东亚文字注音及括号回退 | 现代 |
| `s` | 已不准确或不再相关的内容 | 现代；编辑修订使用 `del` |
| `samp` | 程序输出示例 | 现代 |
| `script` | 嵌入或引用脚本、数据块 | 现代；遵守 CSP 和来源控制 |
| `search` | 搜索或筛选控件区域 | 现代；较新元素，检查目标环境 |
| `section` | 有主题、通常有标题的章节 | 现代；不是通用样式容器 |
| `select` | 选项选择控件 | 现代；需要标签 |
| `slot` | Web Component 插槽 | 现代；用于 Shadow DOM |
| `small` | 附注、免责声明等旁注 | 现代 |
| `source` | 图片、音频或视频候选资源 | 现代；用于 `picture` 或媒体元素 |
| `span` | 无额外语义的行内容器 | 现代 |
| `strong` | 重要、严重或紧急内容 | 现代 |
| `style` | 文档内 CSS | 现代；大型项目通常用外部样式表 |
| `sub` / `sup` | 下标或上标 | 现代；仅用于排版含义 |
| `summary` | `details` 的可见标题 | 现代 |
| `table` | 二维表格数据 | 现代；不用作页面布局 |
| `tbody` / `thead` / `tfoot` | 表格行分组 | 现代 |
| `td` / `th` / `tr` | 表格单元格、表头与行 | 现代；表头设置正确范围 |
| `template` | 不立即渲染的 DOM 模板 | 现代；插入不可信内容仍需处理 |
| `textarea` | 多行文本输入 | 现代；需要标签 |
| `time` | 时间、日期或时长 | 现代；机器值写入 `datetime` |
| `title` | 文档标题 | 现代；每个文档需要有意义标题 |
| `track` | 字幕、说明或章节轨道 | 现代；通常使用 WebVTT |
| `u` | 非文本注释，如拼写标记 | 现代；避免与链接混淆 |
| `ul` | 无序列表 | 现代 |
| `var` | 变量名 | 现代 |
| `video` | 视频播放器 | 现代；提供字幕和替代内容 |
| `wbr` | 可选换行机会 | 现代 |

## 废弃或过时元素
| 元素 | 历史用途 | 现代替代 |
| --- | --- | --- |
| `acronym` | 缩略词 | `abbr` |
| `applet` | Java 小程序 | 不再使用；采用受支持的 Web 技术 |
| `basefont`、`big`、`center`、`font`、`tt` | 字体、字号或布局表现 | 有语义的元素加 CSS |
| `dir` | 目录列表 | `ul` |
| `frame`、`frameset`、`noframes` | 框架页面 | 常规布局；必要嵌入使用受限 `iframe` |
| `marquee` | 滚动文字 | 通常移除；必要动画使用 CSS 并可暂停 |
| `menuitem`、`param` | 菜单命令或插件参数 | 使用按钮、链接；避免旧插件模型 |
| `strike` | 删除线表现 | `s`、`del` 或 CSS |
| `xmp`、`plaintext` | 原样文本 | `pre`、`code` 并正确转义 |

废弃元素即使仍被浏览器容错，也可能缺少互操作性、语义或无障碍支持。
另见：[按功能排序](tags-by-function.md)、[属性参考](attributes.md)。
