# HTML 标签简写及全称

HTML 元素名常来自英文单词或缩写。理解名称来源有助于记忆语义，
但实际用途应以当前 HTML 标准为准，不能只按字面猜测。

## 文档与元数据

| 标签 | 英文全称或来源 | 中文含义 |
| --- | --- | --- |
| `html` | HyperText Markup Language | 超文本标记语言根元素 |
| `head` | head | 文档元数据区 |
| `body` | body | 文档正文 |
| `title` | title | 文档标题 |
| `meta` | metadata | 元数据 |
| `link` | link | 外部资源关系 |
| `base` | base URL | 相对 URL 的基准 |

## 结构与文本

| 标签 | 英文全称或来源 | 中文含义 |
| --- | --- | --- |
| `p` | paragraph | 段落 |
| `h1` 至 `h6` | heading | 一至六级标题 |
| `div` | division | 通用区块容器 |
| `span` | span | 通用行内容器 |
| `nav` | navigation | 导航 |
| `aside` | aside | 补充内容 |
| `em` | emphasis | 语气强调 |
| `strong` | strong importance | 重要内容 |
| `br` | line break | 换行 |
| `hr` | thematic break | 主题分隔 |

## 列表、引用与代码

| 标签 | 英文全称或来源 | 中文含义 |
| --- | --- | --- |
| `ul` | unordered list | 无序列表 |
| `ol` | ordered list | 有序列表 |
| `li` | list item | 列表项 |
| `dl` | description list | 描述列表 |
| `dt` | description term | 被描述的名称 |
| `dd` | description details | 描述内容 |
| `blockquote` | block quotation | 块引用 |
| `q` | quotation | 行内引用 |
| `pre` | preformatted text | 预格式文本 |
| `code` | code | 代码片段 |
| `kbd` | keyboard input | 用户输入 |
| `samp` | sample output | 程序输出示例 |

## 表格与表单

| 标签 | 英文全称或来源 | 中文含义 |
| --- | --- | --- |
| `table` | table | 数据表格 |
| `tr` | table row | 表格行 |
| `th` | table header cell | 表头单元格 |
| `td` | table data cell | 数据单元格 |
| `form` | form | 表单 |
| `input` | input | 输入控件 |
| `label` | label | 控件标签 |
| `fieldset` | field set | 控件分组 |
| `legend` | legend | 分组标题 |

## 媒体与脚本

| 标签 | 英文全称或来源 | 中文含义 |
| --- | --- | --- |
| `a` | anchor | 锚点或链接 |
| `img` | image | 图片 |
| `src` | source（属性） | 资源地址 |
| `iframe` | inline frame | 内联框架 |
| `script` | script | 脚本 |
| `noscript` | no script | 脚本不可用时的内容 |

`frame` 与 `frameset` 是历史标签，已在 HTML5 中废弃，不应用于新页面。
