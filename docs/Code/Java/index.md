# Java 教程

Java 是一门面向对象、跨平台的通用编程语言，常用于后端服务、Android、企业应用、大数据组件和安全工具开发。Java 程序通常编译为字节码，在 JVM 上运行。

## 编译运行

```bash
javac Main.java
java Main
```

单文件源码运行，适用于较新的 JDK：

```bash
java Main.java
```
