# Java 测验

## 选择题

### 1. Java 源文件通常编译成什么？

A. 机器码  
B. 字节码  
C. Shell 脚本  
D. 汇编源码  

答案：B

### 2. Java 程序入口方法通常是什么？

A. `start`  
B. `run`  
C. `public static void main(String[] args)`  
D. `init`  

答案：C

### 3. `HashMap` 存储什么？

A. 纯字符  
B. 键值对  
C. 固定长度数组  
D. 线程  

答案：B

### 4. 哪个关键字用于继承类？

A. `implements`  
B. `extends`  
C. `package`  
D. `throws`  

答案：B

## 实操题

1. 编写一个 `User` 类，包含姓名和年龄。
2. 使用 `ArrayList` 保存整数并排序。
3. 使用 `HashMap` 统计单词出现次数。
4. 使用 `Files.readString` 读取文本文件。
5. 创建一个接口并编写实现类。
