## 可运行实践题

```python
def unique_even(values: list[int]) -> list[int]:
    """返回升序且不重复的偶数。"""
    return sorted({value for value in values if value % 2 == 0})

assert unique_even([3, 2, 2, 4, 1, 6]) == [2, 4, 6]
print("实践题通过")
```
