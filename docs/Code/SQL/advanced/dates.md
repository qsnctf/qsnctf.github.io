# SQL 日期

日期和时间字段常用于订单时间、创建时间、更新时间等场景。

## 常见值格式

- 日期：`'2026-07-17'`
- 日期时间：`'2026-07-17 10:30:00'`

## 示例

```sql
SELECT *
FROM orders
WHERE order_date = '2026-07-17';
```

## 注意点

- 日期函数和类型在不同数据库之间差异较大。
- 生产环境要明确时区和精度。
- 对日期时间做范围筛选时，要注意结束边界写法。
