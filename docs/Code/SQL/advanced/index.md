# SQL 高级教程

本部分继续学习更复杂的查询、连接、结构定义、约束和数据库对象。

## 学习目标

- 掌握多表连接与结果集合并。
- 理解约束、索引、视图等数据库对象。
- 了解建库、建表、修改表结构等 DDL 操作。
- 认识 `NULL`、日期和数据类型的常见处理方式。
