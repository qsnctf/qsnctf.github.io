# SQL 函数

本部分介绍 SQL 中最常见的聚合函数、分组子句和部分字符串、数值、日期函数。

## 学习目标

- 理解聚合函数如何从多行得到一个结果。
- 掌握 `GROUP BY` 和 `HAVING` 的用法。
- 认识常见大小写、截取、长度、时间和格式化函数。
