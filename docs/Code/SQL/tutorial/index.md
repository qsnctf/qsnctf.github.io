# SQL 教程

本部分面向第一次系统学习 SQL 的读者，目标是先掌握最常用的查询、过滤、排序和基础数据修改语句。

## 学习目标

- 理解表、记录、字段和主键这些基础概念。
- 能写出基础 `SELECT` 查询。
- 能使用 `WHERE`、`AND`、`OR`、`ORDER BY` 控制结果集。
- 能完成基础的 `INSERT`、`UPDATE`、`DELETE` 操作。

## 练习建议

使用一个简单的 `users`、`products`、`orders` 表反复练习同一组语句，重点观察：

- 条件变化后结果集如何变化。
- 排序和去重对输出的影响。
- 不带 `WHERE` 的更新和删除为何危险。
