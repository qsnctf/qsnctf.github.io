# SQL 简介

SQL 是 Structured Query Language，即结构化查询语言。它主要用于和关系型数据库交互。

## SQL 用来做什么

- 查询数据：例如找出某个用户或订单。
- 写入数据：插入新记录。
- 修改数据：更新已有字段值。
- 删除数据：移除不再需要的记录。
- 定义结构：创建数据库、表、索引和约束。

## 关系型数据库的基本对象

- 数据库：一组表和其他对象的集合。
- 表：按列组织的数据集合。
- 行：一条记录。
- 列：记录中的一个字段。
- 主键：能唯一标识一行的列或列组合。

示例表：

| id | name | age | city |
| --- | --- | --- | --- |
| 1 | Alice | 20 | Beijing |
| 2 | Bob | 22 | Shanghai |

## 一条最简单的查询

```sql
SELECT name, city
FROM users;
```

这条语句表示：从 `users` 表中取出 `name` 和 `city` 两列。

## 学习 SQL 时要注意

- 先学通用标准语法，再关注具体数据库差异。
- 语句常写成多行，便于阅读和审计。
- 应明确区分“结构操作”和“数据操作”。
- 应使用参数化查询，避免 SQL 注入。
