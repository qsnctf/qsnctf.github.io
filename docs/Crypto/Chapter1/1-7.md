## Ook!

1. **“Ook.”** 表示一个句点
2. **“Ook!”** 表示感叹号

通过这些词组的排列组合，Ook! 语言能够执行与 Brainfuck 相同的功能，但它的语法完全由“Ook!”和不同的变体组合而成

例如：

```
+ -> Ook. Ook!
- -> Ook! Ook.
> -> Ook. Ook.
< -> Ook! Ook!
. -> Ook. Ook. Ook.
, -> Ook! Ook! Ook!
[ -> Ook. Ook. Ook!
] -> Ook! Ook. Ook.
```

**Ook! 编码特点：**

```
Ook. 和 Ook! 组成
```
