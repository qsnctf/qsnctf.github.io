# Sourcetree

Sourcetree 是一款图形化 Git 客户端，适合刚开始学习 Git 或需要可视化查看分支历史的用户。

## 常见功能

- 克隆仓库
- 查看文件改动
- 暂存和提交
- 创建和切换分支
- 拉取、推送、合并
- 查看提交图

## 克隆仓库

在 Sourcetree 中选择 Clone，输入远程仓库地址和本地目录即可。

## 提交代码

1. 查看工作区文件变化。
2. 勾选需要暂存的文件。
3. 填写提交信息。
4. 点击 Commit。

## 推送代码

提交完成后，点击 Push，把本地提交推送到远程仓库。

## 适合和不适合

适合：

- 初学者理解分支和提交图。
- 日常查看历史和处理简单合并。

不适合：

- 自动化脚本。
- 复杂批量操作。
- 需要精确控制参数的高级场景。

建议学习 Sourcetree 的同时，也掌握基本命令行操作。
