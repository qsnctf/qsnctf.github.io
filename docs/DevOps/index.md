# 运维/DevOps

运维与 DevOps 关注系统部署、交付、监控、自动化和运行稳定性。对 CTF 与安全学习来说，容器、镜像、网络、日志和编排能力也常用于搭建靶场、复现实验环境和隔离服务。

4. [Docker 镜像管理与底座](Docker/image-management-and-foundation.md)
5. [容器生命周期与核心操作](Docker/container-lifecycle-and-operations.md)
6. [Dockerfile 进阶与构建艺术](Docker/dockerfile-advanced-builds.md)
7. [数据持久化与网络矩阵](Docker/storage-and-networking.md)
8. [多容器编排与集群生态](Docker/compose-and-cluster.md)

学习 Docker 时建议同时关注三个层面：命令怎么用、底层机制为什么这样工作、生产环境中哪些默认行为需要显式约束。
