## XORTools 隐写

打开文件全是数据且文件头尾不和任何类型的文件头尾有相似的文件都是文件数据被异或加密了

下载工具 Xortools 解密

```python
pip3 install xortool
```

解密这里有问题，应该是 `GoodLuckToYou`

![](https://pic1.imgdb.cn/item/67e9218b0ba3d5a1d7e6c229.png)

使用脚本解密

```python
key = 'GoodLuckToYou'
flag = ''

with open('1') as f:
    con = f.read()
    for i in range(len(con)):
        flag += chr(ord(con[i]) ^ ord(key[i%13]))
        
f = open('flag.txt', 'w')
f.write(flag)
f.close()
```
