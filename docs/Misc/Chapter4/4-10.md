## Wav 文件 LSB 隐写

下载文件发现是 Wav 文件，题目提示是 LSB 隐写

使用 Wav 文件 LSB 隐写工具：Stegolsb

```shell
-h：隐藏声音文件中的数据
-r：从声音文件中恢复数据
-i text：文件的路径
-s text：隐藏在声音文件中的文件路径
-o text：输出文件的路径
-n integer：使用多少个 LSB [默认值：2]
-b integer：从声音文件中恢复多少字节
```
