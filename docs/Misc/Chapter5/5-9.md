## SQL 时间盲注隐写
SQL 时间盲注原理：

**黑客构造一个使用延时函数的 SQL 查询，延时函数的触发条件与数据库中的数据进行比较**

**如果比较为真，延时函数执行，查询执行时间变长**

**如果比较为假，延时函数不执行，查询执行时间较短**
