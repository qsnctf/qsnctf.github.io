# Morse 编码

## 什么是 Morse 编码？

Morse 编码又称摩尔斯电码，它用短信号和长信号表示字母、数字和符号。常见写法中，短信号写作 `.`，长信号写作 `-`。

CTF 中 Morse 编码经常出现在文本、音频、闪烁图片或二进制序列里。它不是加密，而是一种信号编码。

## 基本规则

- 点 `.` 表示短信号。
- 横 `-` 表示长信号。
- 字母之间通常用空格分隔。
- 单词之间通常用 `/` 或多个空格分隔。

例如：

```text
-.-. - ..-.
```

解码为：

```text
CTF
```

## 示例

### 编码

字符串 `FLAG`：

| 字符 | Morse |
| ---- | ----- |
| F | ..-. |
| L | .-.. |
| A | .- |
| G | --. |

所以：

```text
FLAG -> ..-. .-.. .- --.
```

### 解码

密文：

```text
.... . .-.. .-.. --- / -.-. - ..-.
```

逐组查表：

```text
HELLO CTF
```

## 常见变形

- `.` 和 `-` 被替换成 `0` 和 `1`。
- 短长信号被替换成小写和大写、短线和长线、黑白像素。
- 音频题中短音和长音对应点和横。
- 反转点横后再解码，可能得到另一个有意义结果。

## CTF 识别要点

- 出现大量 `.`、`-`、`/`。
- 字符分组长度通常为 1 到 5。
- 音频中有明显长短信号节奏。
- 图片或动图中有规律闪烁、黑白长短条。
