编写脚本打印迷宫图，因为长度为 70，推测要么 7 * 10，要么 10 * 7

```python
def main():
    maze = "*******+********* ******    ****   ******* **F******    **************"
    print(len(maze))
    eid = maze.index("F")
    ex, ey = -4, 5
    c = (eid - ey) // (-ex)
    r = len(maze) // c
    for i in range(r):
        print(maze[c * i:c * (i + 1)])


if __name__ == "__main__":
    main()
```
