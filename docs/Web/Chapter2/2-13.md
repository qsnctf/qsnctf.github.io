`intval()` 是一个用于将变量转换为整数的函数，常用于数据类型转换和输入验证

PHP 5 中的 `intval()` 函数中科学计数法符号 `e` 无效，只会当作正常字符处理

因此如果输入 100e2 之类的数据，会解释称 100，但后面在执行 +1 时，100e2 是解释称 10000 的，因此此处使用 100e2 绕过，结果如下

```python
?num=100e2
```
