MD5 弱比较之前讲过，重点看 strcmp 函数

strcmp() 函数比较两个字符串，且区分大小写

可以看到当两个字符串相同时返回值是 0

![](https://pic1.imgdb.cn/item/67b09d16d0e0a243d4ffac79.jpg)

但当函数接受到不符合字符串类型的参数就会发生错误，并且也返回 0

所以我们只需要提交一个非字符串类型的参数使其返回 0，比如使用数组类型

构造 payload

```php
?v1=QNKCDZO&v2=s878926199a&v3[]=66
```
