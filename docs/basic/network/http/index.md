# HTTP 教程

HTTP 是 Web 通信的核心应用层协议。它采用请求-响应模型，客户端发送请求，服务端返回状态、头部和正文。

4. [HTTP 响应头信息](response-headers.md)
5. [HTTP 状态码](status-codes.md)
6. [HTTP content-type](content-type.md)
7. [MIME 类型](mime-types.md)
8. [HTTP2](http2.md)

Web 安全学习中，HTTP 是理解参数、Cookie、会话、跨站请求和缓存行为的基础。
