# DNS 协议

DNS（Domain Name System）把域名解析为 IP 地址，也保存邮件、服务发现和文本记录等信息。常见端口是 UDP/TCP 53。

## 常见记录类型

| 类型 | 用途 |
| --- | --- |
| A | 域名到 IPv4 地址 |
| AAAA | 域名到 IPv6 地址 |
| CNAME | 别名 |
| MX | 邮件交换服务器 |
| NS | 权威 DNS 服务器 |
| TXT | 文本记录 |
| PTR | 反向解析 |

## UDP 与 TCP

普通查询常用 UDP 53。区域传送、大响应或部分安全扩展场景可能使用 TCP 53。

## 安全注意

DNS 明文查询会泄露访问意图。常见安全问题包括缓存投毒、错误配置的区域传送和恶意域名解析。CTF 中常见任务包括查询 TXT 记录、分析 DNS 隧道和恢复子域名。
