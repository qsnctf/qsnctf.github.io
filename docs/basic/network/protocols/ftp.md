# FTP 协议

FTP（File Transfer Protocol）用于文件传输，传统默认控制连接端口是 TCP 21。

## 基本特点

- 使用控制连接发送命令。
- 使用数据连接传输目录列表和文件内容。
- 传统 FTP 明文传输用户名、密码和数据。
- 主动模式和被动模式的数据连接方向不同。

## 常见命令

| 命令 | 含义 |
| --- | --- |
| `USER` | 用户名 |
| `PASS` | 密码 |
| `LIST` | 列目录 |
| `RETR` | 下载文件 |
| `STOR` | 上传文件 |
| `QUIT` | 退出 |

## 安全注意

传统 FTP 不加密。真实环境应优先使用 SFTP 或 FTPS。CTF 流量题中，FTP 明文凭据和文件内容经常可以直接从抓包中恢复。
