# HTTPS 协议

HTTPS 是 HTTP over TLS。它在 HTTP 与 TCP 之间加入 TLS，提供加密、完整性和服务端身份认证。

## HTTPS 解决的问题

- 防止中间人直接读取明文内容。
- 检测传输过程中被篡改的数据。
- 通过证书验证服务端身份。

## 基本过程

1. TCP 建立连接。
2. TLS 握手协商版本、套件和密钥。
3. 验证服务端证书和主机名。
4. 在 TLS 加密通道中传输 HTTP。

## 安全注意

不要关闭证书验证来“解决”连接错误。证书过期、主机名不匹配或链不可信都应被调查。CTF 中可能通过错误证书、SNI、抓包和代理配置考察 HTTPS 行为。
