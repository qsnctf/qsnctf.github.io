# IMAP 协议

IMAP（Internet Message Access Protocol）用于邮件客户端访问和同步服务器上的邮件。常见端口是 TCP 143，IMAPS 常用 TCP 993。

## 与 POP3 的差异

| 项目 | POP3 | IMAP |
| --- | --- | --- |
| 主要模式 | 下载邮件 | 服务器端同步 |
| 多设备体验 | 较弱 | 较好 |
| 文件夹支持 | 较弱 | 较强 |
| 常见端口 | 110/995 | 143/993 |

## 安全注意

IMAP 登录和邮件内容应通过 TLS 保护。抓包分析时可关注登录、选择邮箱、检索邮件和附件编码。
