# 常见网络设备

网络设备负责连接不同主机、链路和网络。理解设备工作层次有助于判断流量为什么能到达、为什么被转发或为什么被过滤。

## 网卡

网卡负责收发链路层帧，拥有 MAC 地址。操作系统通过网卡驱动把帧交给协议栈。

## 集线器

集线器工作方式接近物理层广播，会把信号转发到所有端口。现代网络中较少使用。

## 交换机

交换机通常根据 MAC 地址表转发以太网帧。它在局域网内减少无关广播流量。

常见概念：

- VLAN：把同一物理交换网络划分为多个逻辑广播域。
- MAC 地址表：记录 MAC 地址与端口的关系。
- 广播帧：目的 MAC 为广播地址，会在广播域内传播。

## 路由器

路由器根据 IP 地址和路由表转发数据包，连接不同网络。家庭网关、企业出口和骨干路由器都属于路由设备。

## 防火墙

防火墙按规则允许或阻止流量。规则可基于 IP、端口、协议、连接状态和应用层特征。

## NAT 设备

NAT 会改写地址或端口，让私有网络主机共享公网出口。它会影响连接方向、端口映射和抓包位置。

## 代理

代理位于客户端和服务端之间，代表一方转发请求。HTTP 代理、SOCKS 代理和反向代理在 Web 场景中常见。

## 负载均衡

负载均衡设备把请求分配到多个后端服务。分析问题时要注意同一域名后面可能有多个真实服务器。
