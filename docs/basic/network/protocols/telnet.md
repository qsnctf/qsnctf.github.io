# Telnet 协议

Telnet 是早期远程终端协议，默认端口通常是 TCP 23。

## 特点

- 交互方式简单。
- 协议本身不加密。
- 用户名、密码和命令内容可被明文抓取。

## 使用场景

现代生产环境不应使用 Telnet 进行远程管理。它更多出现在旧设备、实验环境和 CTF 题目中。

## 与 SSH 的区别

| 项目 | Telnet | SSH |
| --- | --- | --- |
| 加密 | 无 | 有 |
| 默认端口 | 23 | 22 |
| 认证保护 | 弱 | 支持密码、公钥等 |
| 推荐用途 | 实验和兼容旧系统 | 安全远程管理 |
