# TLS 协议

TLS（Transport Layer Security）为应用层协议提供加密、完整性和身份认证。HTTPS、IMAPS、POP3S 等都可以基于 TLS。

## TLS 提供什么

- 服务器证书验证。
- 密钥协商。
- 加密通信。
- 完整性保护。
- 可选客户端证书认证。

## TLS 1.2 与 TLS 1.3

TLS 1.3 简化了握手流程，移除了许多旧算法和不安全选项，通常更安全且更快。

## 抓包分析

即使看不到加密后的 HTTP 内容，也常能观察：

- SNI 主机名。
- 证书链。
- TLS 版本。
- 加密套件。
- 握手是否失败。

不要关闭证书验证来绕过错误，应理解错误原因。
