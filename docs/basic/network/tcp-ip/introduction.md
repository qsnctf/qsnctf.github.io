# TCP/IP 介绍

TCP/IP 协议族支撑了大多数现代网络通信。名字来自其中两个核心协议：IP 负责寻址和跨网络转发，TCP 负责可靠字节流传输。

## TCP/IP 模型

| 层次 | 代表协议 | 作用 |
| --- | --- | --- |
| 应用层 | HTTP、DNS、SMTP | 应用语义 |
| 传输层 | TCP、UDP | 进程到进程通信 |
| 网络层 | IP、ICMP | 主机到主机寻址和路由 |
| 网络接口层 | Ethernet、ARP | 本地链路传输 |

## 为什么重要

CTF Web、流量分析、内网渗透实验和取证都离不开 TCP/IP。理解分层后，可以更快判断问题发生在 DNS、路由、连接、TLS 还是应用协议。
