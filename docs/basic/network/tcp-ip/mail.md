# TCP/IP 邮件

电子邮件通常涉及多个 TCP/IP 应用层协议。

## 常见协议

| 协议 | 作用 | 常见端口 |
| --- | --- | --- |
| SMTP | 发送和转发邮件 | 25、587、465 |
| POP3 | 下载邮件 | 110、995 |
| IMAP | 同步邮件 | 143、993 |
| DNS MX | 查找邮件服务器 | 53 |

## 邮件发送流程

1. 客户端通过 SMTP 向提交服务器发送邮件。
2. 服务器查询收件域名的 MX 记录。
3. 发送方服务器通过 SMTP 转发给接收方服务器。
4. 收件人通过 IMAP 或 POP3 读取邮件。

## 安全关注

邮件安全依赖 TLS、认证、SPF、DKIM、DMARC 和反垃圾策略。CTF 中常见分析点包括邮件头、附件编码、发件路径和 DNS 记录。
