# TCP/IP 协议

TCP/IP 协议族包含多个协作协议。

## 核心协议

| 协议 | 作用 |
| --- | --- |
| IP | 寻址与路由 |
| TCP | 可靠字节流 |
| UDP | 无连接数据报 |
| ICMP | 差错报告和诊断 |
| ARP | IPv4 地址到 MAC 地址解析 |
| DNS | 域名解析 |

## 典型访问流程

访问 `https://example.com/` 可能经历：

1. DNS 查询域名对应 IP。
2. ARP 查询网关 MAC。
3. TCP 与服务器 443 端口建立连接。
4. TLS 握手验证证书并协商密钥。
5. HTTP 在 TLS 通道中传输请求和响应。

协议分析应按链路和时间顺序还原完整流程。
