# 练习CTF

## 如何学好CTF？

!!! note "名言警句"

    学如逆水行舟，不进则退。

学习 CTF 并不难，但真正想学好却很不容易。或许当你逐渐深入这个圈子，你会发现身边的人多少都知道 CTF 是什么。有人在某个方向里钻研得很深，也有人各个方向都略知一二，却始终难以做到精通。真正能在多个领域中游刃有余、被称为“全栈”的人，始终只是极少数。

实际上，CTF 涉及的知识面极广：从Misc、Re、PWN、Web到Crypto，每一个方向都像是一片独立的海洋。想要成为高手，就必须不断在不同的领域中探索、试错、积累。更多时候，你会发现掌握技能不是最难的，难的是坚持学习的耐心、面对失败的勇气，以及不断突破自我的决心。

但正是这种不断挑战未知的过程，让 CTF 充满魅力。每次做出一道题、攻克一个漏洞，你都会感受到成长的喜悦。也许你不一定要成为“全栈大佬”，因为团队的力量有时候更为强大，但只要你愿意保持热情、持续深入，总有一天会在属于你的领域里发光。

我曾经也是这样，是一个各个方向略知一二但达不到精通的人，所以我发自内心的想去做好我们当前这个项目，能够帮助我`吾日三省吾身`，能够将学会的知识点记录下来，不会的知识点去钻研进去，不懂的技术学习成为自己的技术。

    我们已经有了现有的CTF练习平台，但是很多小伙伴都反馈说找不到一个合适的学习成长路线，我的想法比较简单，依托现有的练习平台、靶场、社区，融汇贯通当前文库的知识体系，将知识体系再归纳到题目进行闭环。这样大家在看文档学习知识技术或者看相关视频学习技术知识的时候有题可练，有社区可讨论。
    
    当然经过2018年到现在，很多师傅也随着青少年CTF（中学生CTF）已经到了工作养家的地步了，但是大部分师傅们都还是保留着打CTF的习惯，喜欢CTF的氛围和乐趣，我们也欢迎各位师傅一起共同共建我们的文库，这也是为什么我们底部版权留下了“社区驱动”这样的关键内容。

## CTF练习平台推荐

!!! tip "注意"

    以下排名不分先后

- 青少年CTF练习平台

  - 平台地址：https://www.qsnctf.com
  - 平台特点：为青少年网络安全学习设计，拥有基础题目题型且难度不高，但也有部分赛题复现、社区投稿题目和漏洞复现题型，注重基础技能培养，分类较高，适合初学者入门与综合学习。




- BUUCTF

  - 平台地址：https://buuoj.cn/
  - 平台特点：题库丰富，覆盖Web、PWN、Crypto、Reverse等主流CTF方向；题目难度分层（从入门到进阶），社区活跃，适合系统化训练。




- CTFHub

  - 平台地址：https://www.ctfhub.com
  - 平台特点：提供全栈CTF题目（含Web、PWN、Misc、Reverse、Crypto），题解详细且步骤清晰。




- 攻防世界

  - 平台地址：https://adworld.xctf.org.cn/home/index
  - 攻防世界是集实训、竞赛、办赛、社区于一体的在线网安能力提升平台，致力于为广大网络安全爱好者提供赛题资源与竞赛服务，营造专业、有趣的网络安全学习环境。




- Bugku

  - 平台地址：https://www.bugku.com
  - 平台特点：老牌CTF平台，题目数量庞大（含历史经典题），涵盖基础到进阶内容；提供解题视频和社区讨论区，适合巩固基础技能；界面简洁，适合快速刷题。




- NSSCTF

  - 平台地址：https://www.nssctf.cn/index
  - 平台特点：聚焦实战化竞赛，题目难度分层（初级、中级、高级）。




- KCTF

  - 平台地址：https://ctf.kanxue.com/
  - 平台特点：看雪CTF（简称KCTF）是圈内知名度最高的技术竞技，从原CrackMe攻防大赛中发展而来，采取线上PK的方式，规则设置严格周全，题目涵盖Windows、Android、iOS、Pwn、智能设备、Web等众多领域。




- CTFShow

  - 平台地址：https://www.ctfshow.com
  - 平台特点：目更新频率高（每周新增），涵盖Web、PWN、Reverse、Misc等方向；环境真实，配套解题视频和社区讨论；适合快速提升实战技能。
