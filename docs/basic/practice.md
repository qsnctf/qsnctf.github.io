# 练习CTF

## 如何学好CTF？

!!! note "名言警句"

    学如逆水行舟，不进则退。

学习 CTF 并不难，但真正想学好却很不容易。或许当你逐渐深入这个圈子，你会发现身边的人多少都知道 CTF 是什么。有人在某个方向里钻研得很深，也有人各个方向都略知一二，却始终难以做到精通。真正能在多个领域中游刃有余、被称为“全栈”的人，始终只是极少数。

实际上，CTF 涉及的知识面极广：从Misc、Re、PWN、Web到Crypto，每一个方向都像是一片独立的海洋。想要成为高手，就必须不断在不同的领域中探索、试错、积累。更多时候，你会发现掌握技能不是最难的，难的是坚持学习的耐心、面对失败的勇气，以及不断突破自我的决心。

但正是这种不断挑战未知的过程，让 CTF 充满魅力。每次做出一道题、攻克一个漏洞，你都会感受到成长的喜悦。也许你不一定要成为“全栈大佬”，因为团队的力量有时候更为强大，但只要你愿意保持热情、持续深入，总有一天会在属于你的领域里发光。

我曾经也是这样，是一个各个方向略知一二但达不到精通的人，所以我发自内心的想去做好我们当前这个项目，能够帮助我`吾日三省吾身`，能够将学会的知识点记录下来，不会的知识点去钻研进去，不懂的技术学习成为自己的技术。

!!! note "编者寄语"

    我们已经有了现有的CTF练习平台，但是很多小伙伴都反馈说找不到一个合适的学习成长路线，我的想法比较简单，依托现有的练习平台、靶场、社区，融汇贯通当前文库的知识体系，将知识体系再归纳到题目进行闭环。这样大家在看文档学习知识技术或者看相关视频学习技术知识的时候有题可练，有社区可讨论。
    
    当然经过2018年到现在，很多师傅也随着青少年CTF（中学生CTF）已经到了工作养家的地步了，但是大部分师傅们都还是保留着打CTF的习惯，喜欢CTF的氛围和乐趣，我们也欢迎各位师傅一起共同共建我们的文库，这也是为什么我们底部版权留下了“社区驱动”这样的关键内容。

6. **循环往复，持续成长**
