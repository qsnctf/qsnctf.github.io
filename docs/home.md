# 欢迎来到青少年CTF文库

## 前言

青少年CTF文库旨在为正在学习网络安全、热爱信息攻防、准备参加 CTF 竞赛的你，提供一个 **结构化、系统化、易理解、可实战** 的知识库。  

!!! info "Tips"

    无论你是刚刚接触信息安全的“小白”，还是准备深入某个方向的进阶选手，都可以在这里找到属于你的学习路径。

文库会持续更新最前沿、最常用的 CTF 知识体系，包括：

- Misc、Crypto、Web、Reverse、Pwn的基础知识、考察点
- Web 安全基础与常见漏洞拆解  
- 密码学基础与专题难点解析  
- Pwn 与二进制利用的环境、思路与技巧  
- 逆向工程的工具、流程与题目拆解  
- Misc 杂项方向的技巧与典型题型  
- CTF 实战经验、团队协作与比赛心态  
- 在线工具、通用命令、靶场与实战环境  

欢迎你将这里作为你的 *网络安全学习主基地*。
